CO : Satisfiability Modulo Theories: Introduction and Applications

Date/Time: 2015-05-23 07:00pm Location:

Abstract:

a ∧ ¬(b ∨ a). Is there some assignment to a and b that makes the expression true? This is the well known problem of Boolean satisfiability (SAT). This problem can be extended to predicates over non-Boolean values. For example, is the following satisfiable? (c < d) ∧ (d ≤ c + 1). This extended problem is known as Satisfiability Modulo Theories (SMT). In this talk, I will present an introduction to SMT and its many applications, from scheduling to static program analysis to dependency resolution. I will end with concrete examples, demonstrating how to use an SMT solver from Haskell to prove assertions about a program.
